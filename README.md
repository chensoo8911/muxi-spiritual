# 木喜身心靈

整理自吳木喜與學生多年分享的身心靈資源網站。分三區：

- **靈性書籍／有聲書** — 可閱讀全文（海奧華預言）、一鍵朗讀，並收錄相關有聲書。
- **靈性／外星人影片** — 依主題分類、輕量嵌入 YouTube（點縮圖才載入）。
- **木喜專區** — 十一項核心教導與十句金句。

## 技術

純靜態 HTML／CSS／JS，無建置步驟，適合 GitHub Pages。

- 字體：Noto Serif TC（小說襯線、大字級，長輩好讀）
- RWD 響應式，手機自動放大字級
- 朗讀：瀏覽器語音合成（Web Speech API），自動挑最自然的中文語音
- 重新產生頁面：`python3 build.py`（需同層級的 `../電子書/` 內容）

## 版權

各書籍、影片版權歸原作者所有，本站僅作靈性學習與紀念之用。
